## 📌 Project Overview
This project leverages Machine Learning to predict whether a customer will subscribe to a term deposit based on demographic and behavioral data. By analyzing over 150,000 banking records from the UCI Bank Marketing dataset, this model helps financial institutions optimize their marketing campaigns, target the right demographics, and improve conversion rates.

## 🚀 Business Value
* **Targeted Marketing:** Identifies high-conversion customer segments.
* **Cost Efficiency:** Reduces wasted marketing spend by focusing only on probable subscribers.
* **Workflow Optimization:** Streamlines the analysis of massive banking datasets to drive data-informed product strategies.

## ⚙️ Methodology & Architecture

1. **Data Preprocessing:** Cleaned and engineered features from 150k+ records. Handled missing values and encoded categorical variables.
2. **Exploratory Data Analysis (EDA):** Generated visual insights to uncover customer behavior patterns.
3. **Model Development:** Built and trained a **Decision Tree Classifier** to map decision paths based on customer attributes.
4. **Evaluation:** Tested and optimized the model using hyperparameter tuning to ensure robust performance.

## 📈 Results
* **Accuracy:** Achieved **~87% prediction accuracy** for financial product subscriptions.
* Successfully mapped key behavioral indicators that strongly correlate with term deposit purchases.
