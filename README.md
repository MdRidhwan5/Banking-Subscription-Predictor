## ⚙️ Technical Architecture & Workflow
The solution utilizes an automated data pipeline to clean, engineer, and feed data into an ensemble ML architecture.

**The Workflow:**
`Data Collection` ➔ `Data Cleaning` ➔ `EDA` ➔ `Feature Engineering` ➔ `Model Training` ➔ `Evaluation` ➔ `Production Prediction`

* **Ensemble ML Models:** Leveraged robust decision-tree based ensembles to handle the highly imbalanced nature of the dataset.
* **Feature Engineering:** Engineered behavioral and demographic features to capture high-intent user signals.
* **Hyperparameter Tuning:** Optimized tree depth and entropy criteria to ensure model scalability and reliability.
